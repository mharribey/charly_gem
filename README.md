# CharlyDev

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'charly_dev'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install charly_dev

## Usage

Just type this in terminal and let the power of the gem blow your mind :

    $ charly_dev
